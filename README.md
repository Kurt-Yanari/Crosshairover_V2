# Crosshairover_V2
Crosshair for games
crosshair_singlefile.py — PyQt5 Crosshair Overlay
Особенности:
- Cross / Dot режим
- Автоцентровка прицела при изменении окна
- Настройки сохраняются в crosshair_settings.json
- Горячие клавиши:
    F8 — показать/скрыть панель
    F9 — включить/выключить click-through
    Esc — закрыть
Зависимости: pip install PyQt5 keyboard
